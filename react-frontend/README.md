# FinSage React Frontend

React + Vite frontend for FinSage parser/analytics flow.

## Run backend API

```bash
cd /path/to/ET-AI-Hackathon26
source ../.venv/bin/activate
uvicorn api.react_api:app --host 127.0.0.1 --port 8000 --reload
```

## Run frontend

```bash
cd react-frontend
npm install
npm run dev
```

Open: http://127.0.0.1:5173

## Build check

```bash
npm run build
```
